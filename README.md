# Aplicativo de Orçamento Pessoal

Este aplicativo web ajuda a controlar suas despesas. Você pode adicionar novas despesas e ver uma lista das que já foram adicionadas.

## Funcionalidades

* **Adicionar Despesas:** Salve os detalhes de cada gasto (data, tipo, descrição, valor).
* **Ver Despesas:** Veja todas as despesas em uma lista, com opção de filtrar.
* **Filtrar:** Encontre despesas por data, tipo, descrição ou valor.
* **Excluir:** Remova despesas da lista.

## Como Usar

1.  **Adicionar:**
    * Abra `index.html` no navegador.
    * Preencha o formulário e clique em "Cadastrar".

2.  **Ver/Filtrar:**
    * Abra `consulta.html` no navegador.
    * Use os campos para filtrar e veja a lista de despesas.
    * Clique no "x" para excluir uma despesa.

## Arquivos

* `index.html`: Para adicionar despesas.
* `consulta.html`: Para ver e filtrar despesas.
* `app.js`:  Código do aplicativo.

## Tecnologias

* HTML, CSS (Bootstrap), JavaScript
* Dados salvos no navegador (localStorage).
